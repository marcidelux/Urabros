Detailed Description

Main logic part of the framework. There are four important parts of this source file.

Author
marton.lorinczi
Date
Aug 10, 2020
See also
UrabrosInit()
Creates the uTasks array where all of the tasks pointers are stored. Calls all the necessary inits for communication handlings. Creates the three main threads.
urabrosCommunicationFunction()
Thread responsible for handling the incoming messages. It can adds commands to thecommand list, starts the tasks, delete command form the list, and stop the tasks.
urabrosLogicControlFunction()
This thread updates the statuses of the active commands, based on the statuses of the running tasks.
urabrosMessageSenderFunction()
This function is responsible for sending any kind of messages to the controller PC.
