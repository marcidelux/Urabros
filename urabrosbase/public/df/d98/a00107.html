Detailed Description

This header file contains the main functions an Urabros_TaskTypeDef has.
Include this header to a Tasks source file, don't include it to it's header!
All functions described here are inline functions, thats how every separate
uTask will have it's own copy of the function.
It has an important static variable called Task, with including this header
every task will has it's own handler. It has to be asigned to the actual task's
Urabros_TaskPtrTypeDef extern variable what is in the actual task's header.
