 UrabrosTimeout.hThis header contains static functions for Timeout checks, based on HAL_GetTick() function.
Include this library to any of the Urabros Tasks source or Urabros Modules source.
Each file will have its own copy of the static functions.
Before including it to a source file you have to define some preprocessor defines
#TIMEOUT_ENABLED - Define this to 1 for enable timeout functions or 0 to disable them
TIMEOUT_INSTANCES - Define this with a number of instances you would like to use it.
