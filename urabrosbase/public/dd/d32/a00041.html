enum Urabros_CommandReturnStatus

An enum for the return status of the processed command.

Enumerator
uCommandOk 

Everything is fine.

uCommandAdded 

Command added to the command list with the given Task ID.

uCommandNotFinished 

Task is still running, can't execute command.

uCommandNotFound 

Task ID is not found on the command list.

uCommandDeleted 

Task is deleted from the command list.

uCommandTimedOut 

Can't execute command it timed out.

uCommandOwerFlow 

The command list is full, or the Tasks Queue is full.

uCommandIdAlreadyUsed 

The given ID is already on the command list.

uCommandIdOutOfRange 

The given task ID is not valid.

uCommandCantReceiveData 

Task is finished or waiting for start, at this point It cannot receive data.

uCommandError 

Other error.
