◆ Urabros_CommandIdTypedef

typedef uint8_t Urabros_CommandIdTypedef

This is an Urabros command typedef, even if it is an uint8_t this what is secured to not mess it up in the code.


Every high level jobs has to own one command id, and connected it to its own uTask.
Leave the TASK_ID_NON as 0 and TASK_ID_TEST as 255, so the user defined task can be between 1 - 254.
